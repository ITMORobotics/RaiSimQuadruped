# RaiSimQuadruped

## Setup

```
sudo chmod +x installRaiSimLib.sh
```
If you would like add path in **~/.bashrc** or **~/.zshrc** automaticly, add key.**--addpath**
```
./installRaiSimLib.sh 
# ./installRaiSimLib.sh --addpath
```
